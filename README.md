## üöÄ Fitur <a id="fitur"></a>

| Modul                  | Deskripsi                                                            |
| ---------------------- | -------------------------------------------------------------------- |
| **Diagram Interaktif** | Menampilkan hierarki struktur organisasi secara visual dan dinamis   |
| **Detail Jabatan**     | Menampilkan informasi lengkap setiap posisi/jabatan dalam organisasi |
| **Pencarian & Filter** | Mencari jabatan atau personel tertentu dengan cepat                  |
| **Responsive Layout**  | Tampilan optimal di desktop maupun perangkat mobile                  |
| **Mode Gelap/Terang**  | Dukungan Dark Mode otomatis sesuai preferensi perangkat              |
| **Integrasi API**      | Data struktur diambil dari API endpoint untuk memudahkan pembaruan   |

## üõ†Ô∏è Instalasi <a id="instalasi"></a>

```bash
# Klon repo
$ git clone https://github.com/BhinnekaDev/STRUKTUR-ORGANISASI-STASIUN-KLIMATOLOGI.git
$ cd STRUKTUR-ORGANISASI-STASIUN-KLIMATOLOGI

# Instal dependensi
$ npm install
```

Buat file **`.env`** (jika ada konfigurasi env):

```env
NEXT_PUBLIC_API_URL=https://api.kamu.com/api
```

Jalankan mode development:

```bash
$ npm run dev
```

Akses **[http://localhost:3000](http://localhost:3000)**
